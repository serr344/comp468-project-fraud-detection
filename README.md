# COMP468 - Project

## Credit Card Fraud Detection

---

## Project Status
This project is currently under development.

---

## Project Objective
The objective of this project is to build a Machine Learning model that can detect fraudulent credit card transactions.

The model will classify each transaction as either normal or fraudulent by learning patterns from transaction and identity-related features.

---

## Dataset
Dataset used in this project:

**IEEE-CIS Fraud Detection Dataset**

Source:  
Kaggle Competition - IEEE-CIS Fraud Detection
